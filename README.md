# JavaScript Core Concepts Practice Portfolio

This repository serves as a dedicated practice sandbox for mastering fundamental JavaScript concepts, DOM manipulation, asynchronous programming, and web API integration. Each project focuses on isolating and implementing specific programming patterns using vanilla HTML5, CSS3, and JavaScript (ES6+).

---

## 📁 Repository Structure Overview

The project environment is broken down into modular structural components across three main technology directories:

```text
├── HTML/                      # Application user interfaces and document markup
│   ├── bookSystem.html        # Book tracking systems structure
│   ├── colorGame.html         # Interactive color puzzle dashboard
│   ├── debugCode.html         # Error diagnostic console UI
│   ├── employeeDetails.html   # HR record listing matrix
│   ├── followersCount.html    # State-tracking analytics window
│   ├── healthArticle.html     # Dynamic research article card layout
│   ├── newsArticle.html       # Content management wireframe
│   ├── productSurvey.html     # Client input form controller
│   ├── restaurantMenu.html    # Menu card array iterator UI
│   ├── speedAnalysis.html     # Computational text processing counter
│   └── todoList.html          # State task management portal

│   ├── colorGame.js           # Math.random RGB hex algorithms
